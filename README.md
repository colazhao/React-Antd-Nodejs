# 项目说明

## 1.reactAntd文件夹

--前端文件

--技术栈：React、antd

```
npm install
npm run start
```



## 2.nodeDemo文件夹

--后端文件

--技术栈：nodejs、express

```
npm run start
```



## 3.nodedb文件

--数据库SQL文件

--数据库类型：Mysql

使用数据库工具导入

## 4.项目功能说明

### 首页

实践react及antd中的Carousel、Statistic、Calendar、Card。呈现显示效果
